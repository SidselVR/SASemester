SmartDrinking

What is "SmartDrinking"?

SmartDrinking is a university project that aims to change people's drinking habbits to be more responsible. While it doesn't aim to try and get anyone to stop drinking (let's be honest, that won't happen), it does aim to change the way others use alcohol and how much they consume in one sitting.
